TrybeWarts

Objetivo do Projeto:
Desenvolvi uma página de formulário para a fictícia Escola de Magia de Trybewarts, permitindo que estudantes enviassem feedbacks. Inspirado por "Harry Potter", criei campos para informações pessoais, casas, matérias, avaliação da escola e comentários, validando os campos antes do envio.
Habilidade Aplicada:
Dominei HTML para estruturar, CSS para estilizar, e JavaScript para interatividade. Usei Git para versionamento, trabalhei em equipe e aprimorei desenvolvimento web e colaboração.

Pixel Art

Objetivo do Projeto:
Nesse projeto, implementei um editor de arte com pixels, permitindo aos usuários escolher cores da paleta e pintar em um quadro. A proposta era criar uma experiência de arte digital envolvente.
Habilidade Aplicada:
Utilizei HTML e CSS para estruturar e estilizar a página. A lógica de seleção de cores, preenchimento de pixels e interações com botões foram desenvolvidas em JavaScript. O LocalStorage foi empregado para salvar e recuperar desenhos e utilizei o Git para versionar o projeto.
Após concluir o projeto, criei um editor de arte digital interativo, combinando design e programação para oferecer uma plataforma criativa aos usuários.
